This directory holds prebuilt wheel packages used during environment setup (see scripts/setup.sh).

Included wheels (cp310, linux_aarch64 / Jetson):

  torch-2.7.0-cp310-cp310-linux_aarch64.whl
  flash_attn-2.7.4.post1-cp310-cp310-linux_aarch64.whl
  triton-3.2.0-cp310-cp310-linux_aarch64.whl

To obtain compatible builds, download wheels from the Jetson AI Lab index:

  https://pypi.jetson-ai-lab.io/

Place downloaded .whl files in this folder (engine/wheel_pkgs) before running setup.
