
# Throughput Evaluation

We provide two ways to run the throughput evaluation:

1. **Remote Access**: connect to our platform and run the evaluation directly.
2. **Local Execution**: run the evaluation on your own device using the provided scripts and code.


## 1. Remote Access

This option is primarily intended for AE reviewers. No hardware setup is required. You can open a Jupyter Notebook in a PC browser and run the evaluation remotely.

After opening the remote environment in your browser, open `run_evaluation.ipynb` and execute the cells in order.

For security reasons, we do not publish the access IP address or password in this repository.  
Please contact us via HotCRP, and we will share the credentials promptly.
